Manage budget
User Stories


1. As a user I want to be able to create a new budget.
2. As a user I want to be presented with suggestions about normal expenses.
3. As a user i want to be able to sort and adapt the suggestions to fit my personal profile.
4. As a user I want to be able to edit my budget.
5. As a user I want to be able to remove my budget.
6. As a user I want to be able to sort my budget values into categories.
Manage account balance
User stories


1. As a user i want to be able to set my account balance.


1. As a user I want to be able to change my set account balance(without making a transaction).
2. As a user i want to be able to register a transaction.
3. As a user i want to be able to see my account history.
Manage categories
User Stories


1. As a user I want to be able to add categories.
2. As a user I want to be able to remove categories, and if i do I want transactions made in that category to be listed as other.
3. As a user I want to be able to change categories.
Compare budget and transactions
User Stories


1. As a user I want to be able to compare my budget and my balance changes.
2. As a user I want to be presented with this data both visually in graphs and numerically.
